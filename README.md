# Omega

Omega is a local desktop media-center project built around library scanning, show/episode organization, playback, shuffle logic, and a polished app interface.

## Status

Initial source upload. Private media libraries, runtime caches, generated thumbnails, local source lists, and user media files are intentionally excluded from Git.

## Privacy / Size Rules

Do not commit media folders, video/audio/subtitle files, generated thumbnails, cache folders, local library state, secrets, or environment files.
